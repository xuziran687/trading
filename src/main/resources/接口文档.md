# 二手交易平台 Longan Trade 接口文档（完整版）

## 统一返回格式



```json
{
  "code": 200,
  "msg": "成功",
  "data": {}
}
```

- code：200 = 成功，其他 = 失败
- msg：提示信息
- data：业务数据

------

# 一、用户模块（User）

## 1. 注册

- **POST /api/user/register**
- 功能：手机号注册，自动创建钱包、信用记录

**请求参数**



```json
{
  "phone": "13800138000",
  "password": "123456",
  "nickname": "小明"
}
```

**响应示例**



```json
{
  "code": 200,
  "msg": "注册成功",
  "data": {
    "userId": 1001,
    "nickname": "小明",
    "phone": "13800138000",
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
  }
}
```

# 文档说明

- 所有接口均支持分页：page、size
- 所有需要登录的接口，请求头必须带：`Authorization: Bearer {token}`
- 所有时间格式：`yyyy-MM-dd HH:mm:ss`
- 所有金额类型：DECIMAL (12,2)
